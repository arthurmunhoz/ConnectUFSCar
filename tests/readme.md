# Testes

Pasta de testes.<br/>
Scripts feitos para rodar com o MonkeyRunner.<br/>
Para executar, substitua a linha `device.installPackage('apkDir')` por `device.installPackage('diretório\\do\\apk\\')` (Por exemplo, `device.installPackage('/home/Usuário/AndroidStudioProjects/ConnectUFSCar-master/app/build/outputs/apk')`)<br/>
Abra a pasta sdk do Android (geralmente localizada em `C:\Users\Usuário\AppData\Local\Android\sdk`);<br/>
Abra a pasta tools na janela de comandos;<br/>
Execute o comando `monkeyrunner.bat diretório/do/seu/script.py`.
